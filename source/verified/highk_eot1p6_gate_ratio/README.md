# High-K EOT 1.6 nm + Gate Ratio + Reliable DIBL

학회 발표의 High-K 및 gate-ratio analysis에 사용한 대표 성공 code set입니다.

## Main Features

- SiO₂ IL 0.5 nm + HfO₂ 5.64 nm
- EOT ≈ 1.6 nm, physical thickness 6.14 nm
- normalized `DMG_RatioS` parameter
- moving GateS/GateD contact centers
- gateS/gateD NonLocal tunneling
- corrected Vtgm with `−Vd/2`
- constant-current CC1/CC2 threshold and valid flags
