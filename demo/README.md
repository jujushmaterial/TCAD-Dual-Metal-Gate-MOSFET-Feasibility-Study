# Project Page Design Demo

이 폴더는 기존 프로젝트 문서와 독립된 상세 페이지 디자인 시안입니다.

## 확인 방법

1. `design/project-page-v2` 브랜치 ZIP을 다운로드합니다.
2. 압축을 풉니다.
3. `demo/project-page.html`을 Chrome에서 엽니다.
4. 상단 버튼, 내부 목차, 본문 이미지·표, 하단 자료 목록과 `코드 보기` 모달을 확인합니다.

인터넷 연결이 있어야 Google Fonts, Pretendard와 기존 GitHub Pages 이미지가 표시됩니다.

## 안전 범위

- 기존 `index.md` 수정 없음
- 기존 README 및 연구 문서 수정 없음
- 기존 source, results, presentation, figures 수정 없음
- `demo/` 폴더만 추가
