## Data Policy

- 학회 발표 결과를 최종 narrative baseline으로 사용합니다.
- 초기 값은 삭제하지 않고 original workbook과 evolution context에 보존합니다.
- low-confidence DIBL outlier는 명시적으로 표시합니다.
- High-K Ig는 calibrated absolute prediction이 아닌 relative first-pass comparison입니다.
